How to setup your SSH on NixOS

  • NixOS
  • SSH

SSH configuration

TO change the confguration of you SSH, you have to edit your configuration.nix.
You cannot edit the default ssh file located in /etc/ssh/sshd_config.

Enable openssh

First, you need to enable the OpenSSH Daemon.

[...]
# List services that you want to enable:
services.openssh.enable = true;
[...]

Enable root login

For your first install, you may need to acces your NixOS VM using SSH but the root connction is not allowed by default.
To authorize ots connection, in a temporary way of course, you need to add this line into your configuration file :

# Activer Root login
services.openssh.permitRootLogin = "yes";

For more informations : Nixos-permitroot

Enable SSH in the firewall

# Open port in the firewall
networking.firewall.allowedTCPPorts = [ 22 ];

Use public key !

Create your user.
You must add your public key in order to log in to your NixOS.

# Don't forget to assign a password with the command `passwd` !
users.users.eidorian = {
  isNormalUser = true;
  extraGroups = [ "wheel" ]; # Enable ‚Äòsudo‚Äô.
  openssh.authorizedKeys.keys = ["Add your public key here"];
};

Use another file !

To make your configuration cleaner, you can create an ssh.nix file in the same place as your configuration.nix file and add everything we’ve just seen here.

This way, it will be easier for you to search for your settings than searching a single large file.

For the ssh.nix file to be recognized by the system and by the configuration.nix file, you have to tell it in the imports section.

imports =
[ # Include the results of the hardware scan.
        ./hardware-configuration.nix
        ./ssh.nix
];
