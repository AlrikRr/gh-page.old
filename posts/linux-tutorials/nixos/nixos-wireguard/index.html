How to setup Wireguard NixOS

Special thnks to Rayan Lahfa
and Klaes ;)

Wiki !

You can use the default configuration file : Nixos-Wireguard

Install & setup

First, you need to install wireguard using :

$ nix-env -iA nixos.wireguard

Now, you need to create the wireguard.nix file, this file will contain the default configuration of Wireguard :

$ vim /etc/nixos/wireguard.nix

Generate keys

You need to creat a bunch of public/private keys for your server and your android client :

# Server keys
$ umask 077
$ mkdir /etc/wireguard
$ wg genkey > /etc/wireguard/private
$ wg pubkey < /etc/wireguard/private > /etc/wireguard/public

# Client keys
$ mkdir -p /tmp/client
$ wg genkey > /tmp/client/private
$ wg pubkey < /tmp/client/private > /tmp/client/public

Don"t forget to edit the configuration file with your keys

Edit configuration.nix

Add the Wireguard file in the imports of the configuration.nix :

imports =
  [ # Include the results of the hardware scan.
    ./hardware-configuration.nix
    ./wireguard.nix
  ];

Rebuild & start wireguard

# Rebuild nixos
$ nixos-rebuild switch

# Reload wireguard
$ systemctl status wireguard-wg0.service
$ systemctl restart wireguard-wg0.service

Connect your Andoird

If you want to use your VPN with you Android device, just follow these steps.

First, if you want to access your VPN from the outside, you must open a port of your Internet Box to redirect it to the default Wireguard port (mentienned into wireguard.nix).

Create the config file for your Android

This file will be read by the wireguard APK.

$ cat conf_phone

[Interface]
PrivateKey = <put-private-key-client>
Address = 10.220.0.2 #IP that will be used by your phone 
DNS = 8.8.8.8

[Peer]
PublicKey = <put-pubkey-server>
Endpoint = <Public IP from your internet box followed by the port you opened>90.10.22.11:48999
AllowedIPs = 0.0.0.0/0
PersistentKeepAlive = 25

Create a QRcode

If you don’t want to bother entering the configuration by hand. With the Wireguard APK it is possible to scan a QRcode !
I will show you how to create a QRcode quickly from a file, here our conf_phone file.

# Install qrencode
$ nix-env -iA nixos.qrencode

# Create the QRcode with conf_phone
$ qrencode -t ansiutf8 < conf_phone

And voilà ! Now you can scan your QRcode with the Wireguard APK and conenct to you VPN !
